## Overview

This project models cross-stablecoin and cross-venue depeg propagation as a node-classification problem on a temporal graph. The approach follows the Uniswap paper's pipeline structure and integrates microstructure signals from the IAQF 2026 codebase.

**Core question**: Given a peg-stress event originating at node _i_ (asset × venue), which nodes _j_ will experience contagion within horizon Δ?

## Node universe

Each node is an **(asset, venue)** pair. The built dataset (`data/processed/dataset_manifest.json`)
uses the two deepest venues with continuous 1-minute history:

- **Assets**: the stablecoins stressed across the seven crises (USDC, USDT, DAI, FRAX, BUSD, UST, …)
- **Venues**: **Binance** (USDT-quoted spot) and **Coinbase** (USD-quoted spot). Binance prices
  are converted to true USD via the Coinbase USDT/USD mid, so a USDT depeg is not hidden by the
  quote currency. (Curve/Uniswap/Kraken ingestion exists in `configs/` but is not used by the
  published 7-episode dataset.)

## Model ladder

| Model | Type | Input |
|---|---|---|
| Majority / Persistence | Baseline | labels |
| Logistic Regression | Linear tabular | node features |
| XGBoost | Nonlinear tabular | node features |
| LSTM | Sequential | per-node feature sequences |
| GraphSAGE | GNN | temporal graph |
| GAT | GNN + attention | temporal graph + edge features |

## Key design decisions

- **6-hour rolling edge window** — matches Uniswap paper edge construction
- **Chronological train/val/test split** across episodes — no shuffle, no leakage
- **PR-AUC** as the headline metric — handles severe class imbalance (price-label AUROC is misleading)
- **Leave-one-crisis-out (LOCO)** — a model never trains and tests on the same crisis
- **Lead-time analysis** — horizons: 30 min, 1 h, 4 h, 24 h (headline 24 h)

## Node features

| Feature | Source |
|---|---|
| Price ratio (vs $1 peg) | IAQF fetch pipeline |
| Realized vol (1 h, 24 h) | IAQF |
| Log volume (1 h) | IAQF |
| Amihud illiquidity | IAQF |
| Kyle's λ | IAQF |
| OU half-life | IAQF |
| LOP wedge (cross-venue) | IAQF |
| TVL (USD) | DeFiLlama |

## Edge features

- Rolling 6-hour price correlation
- Cross-pool flow (USD)
- Lead-lag minutes (permutation-test cross-correlation)
- Shared LP percentage

## Quickstart

```bash
pip install -e ".[dev]"

# 1. Build the real dataset (Binance + Coinbase 1-min, peg-adjusted)
python scripts/build_real_dataset.py

# 2. Model ladder + LOCO at the headline 24h horizon
python eval/run_benchmark.py --horizon 1440

# 3. Multi-seed robustness + graph ablation
python eval/robustness_multiseed.py && python eval/ablation_graph.py

# 4. Tests
pytest tests/
```

Or run the whole pipeline (dataset → ladder → ablation → hubs → figures → tests) with
`bash reproduce.sh`.

## Reproduce (data → analysis → paper)

**Prerequisites.** Python 3.11. For the exact pinned environment use conda — `conda env create -f environment.yml && conda activate scgnn` — or with pip:
```bash
pip install -e .
```

**One shot.** `bash reproduce.sh` runs the full 7-step pipeline below in order and writes
every artifact to `results/` and `exports/`.

### Reproduce the paper's headline numbers

Each claim is regenerated by one committed script writing one committed artifact. Training
is seeded (5-seed LOCO; the edge-rewiring null via `SCGNN_REWIRE_SEED`).

| Paper claim | Command | Output artifact |
| --- | --- | --- |
| Build the real dataset (Binance + Coinbase 1-min, peg-adjusted, leakage-safe) | `python scripts/build_real_dataset.py` | `data/processed/dataset_manifest.json` |
| GAT beats best non-graph by **+0.175** PR-AUC at 24 h (5-seed) — abstract, Tab. `loco_stability` | `python eval/run_benchmark.py --horizon 1440` then `python eval/robustness_multiseed.py` | `results/eval/loco_stability_comparison_h1440.csv`, `loco_all_7_h1440.csv` |
| Four-condition ablation: directed lead-lag **edges add +0.10** PR-AUC — Tab. `ablation` | `python eval/ablation_graph.py` | `results/eval/ablation_4condition.csv` |
| Edge-rewiring null (topology, not edge count, drives the gain; 5 rewirings) | `python eval/run_edge_rewiring_null.py` | `results/eval/edge_rewiring_null.json` |
| Exported per-crisis hub ranking + ABM calibration target | `python interpret/run_interpret.py --episode USDC_SVB --horizon 1440 --kind gat` then `python scripts/export_calibration.py` | `exports/schema_v1.json`, `exports/calibration_v1.csv` |
| All paper figures | `python scripts/generate_figures.py` | `paper/standalone_gnn_paper/figures/*.png` |

The compiled paper is **`paper/standalone_gnn_paper/main.tex → main.pdf`** (build:
`cd paper/standalone_gnn_paper && latexmk -pdf main.tex`).

**Exact reproduction.** Tested with Python 3.11. The committed `results/` tables are the exact published numbers; training is seeded (5-seed LOCO; edge-rewiring null via `SCGNN_REWIRE_SEED`). The real venue dataset under `data/` is gitignored and rebuilt from public Binance + Coinbase data via `scripts/build_real_dataset.py`; live archives may revise, so the committed `results/` artifacts are canonical for the published figures.
